## Bug 清单

### BUG-001: Agent选择器缺少OpenAI Agent [P1]

**发现时间**: 2026-03-19 14:58
**严重程度**: P1 - 功能缺失

**复现步骤**:
1. 打开 http://localhost:3000
2. 点击"开始新对话"
3. 查看动物选择列表

**预期结果**: 显示4个Agent（雪球、六六、小黄、OpenAI Agent）
**实际结果**: 只显示3个Agent（雪球、六六、小黄）

**原因分析**: 前端硬编码了Agent列表，没有从API获取

**修复建议**: 
- 从 `/api/api/animals` 动态获取Agent列表
- 或在前端添加 OpenAI Agent 到配置
