### 1. Agent 记忆系统 (claude-mem 思路)
**来源**: 2026-03-30
**适用产品**: zoo, mood-diary-wechat

为每个 Agent 增加"记忆卡片"：
- 短期记忆：当前会话上下文摘要
- 长期记忆：跨会话积累的偏好/习惯
- 可视化管理：查看/编辑/删除记忆
- 自动触发：处理任务时自动注入相关记忆

### 4. 结构化工作流引擎 (Superpowers 思路)
**来源**: 2026-03-19
**适用产品**: 所有产品

将复杂任务拆分为阶段：
1. 理解阶段：AI 先提问澄清需求
2. 设计阶段：展示方案分块让用户确认
3. 执行阶段：子任务派发 + 结果验证
4. 审查阶段：自动质量检查
5. 完成阶段：总结 + 后续建议
