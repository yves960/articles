# 💡 今日创意 - 2026-03-26

## 发现的产品

### 1. SuperMemory - AI时代的记忆引擎
https://github.com/supermemoryai/supermemory
- 给AI Agents提供快速、可扩展的记忆系统
- 类似于给AI的"第二大脑"

**解决的问题：** AI应用缺乏长期记忆和上下文管理能力，每次对话都是全新的。

**创新点：**
- 提供Memory API for AI era
- 极快的检索速度
- 可扩展的架构

**可借鉴之处：** 
- 我们的产品可以借鉴其记忆层设计，让AI更好地理解用户历史

---

### 2. Claude Subconscious - 给Claude Code一个"潜意识"
https://github.com/letta-ai/claude-subconscious
- 让AI在执行任务时保持持续的上下文和记忆

**解决的问题：** Claude Code等AI工具在长时间任务中会"遗忘"之前的上下文

**创新点：**
- 后台持续运行的记忆层
- 非侵入式地增强AI能力

**可借鉴之处：**
- 可用于 zoo Agent平台的长期记忆设计
