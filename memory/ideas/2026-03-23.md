# 创意发现 - 2026-03-23

## 发现的产品

### 1. deer-flow (bytedance)
https://github.com/bytedance/deer-flow

开源的SuperAgent框架，可自主研究、编码和创建。具备沙箱、记忆、工具、子代理和消息网关，能处理从分钟到小时的不同复杂程度的任务。

**解决的问题**：让AI Agent能够自主完成复杂的多步骤任务

**创新点**：
- 多层代理架构
- 沙箱隔离确保安全
- 记忆系统支持长期任务

### 2. Project N.O.M.A.D (Crosstalk-Solutions)
https://github.com/Crosstalk-Solutions/project-nomad

自包含的离线生存计算机，内置关键工具、知识和AI，随时随地保持信息获取和能力。

**解决的问题**：无网络环境下的生存需求

**创新点**：
- 完全离线可用
- 知识库本地化
- AI能力嵌入式

### 3. browser-use
https://github.com/browser-use/browser-use

让网站对AI代理可访问，轻松自动化在线任务。

**解决的问题**：AI无法操作网页的痛点

**创新点**：
- 通用网页操作接口
- 自动化工作流

## 创意点子

1. **FireworksLife - AI任务自动化**：参考deer-flow的多代理架构，让FireworksLife的用户可以通过自然语言描述复杂任务，AI自动拆解并执行

2. **zoo/offline-mode**：参考Project N.O.M.A.D 的离线概念，为zoo增加离线知识库功能，用户可以下载常用知识模块到本地

3. **mood-diary-wechat - AI自动化分析**：结合browser-use的思想，实现自动汇总和分析用户在其他平台的活动来辅助情绪记录
