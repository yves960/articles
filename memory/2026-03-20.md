## h-agent 核心要点

### 命令速查
```bash
h-agent init                    # 初始化
h-agent start/stop/status       # 后台服务
h-agent team init               # 初始化团队
h-agent team talk <agent> "msg" # 与特定 agent 对话
h-agent web                     # Web UI
h-agent session list/create     # 会话管理
h-agent skill list              # 技能
```

### 重要修复
1. Session 重名：自动生成唯一名字
2. Team talk：存储 prompt，重启重建 handler
3. Skill 注册：自动加载到 agent
4. 人格系统：支持 SOUL.md 格式

### CI 测试
- 跳过需要 opencode CLI 的测试
- 跳过 Windows 测试（路径问题）
- 使用 `@pytest.mark.skipif` 条件跳过
