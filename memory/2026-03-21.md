## h-agent 当前状态

### 核心功能
- 多 Agent 协作（Team）
- Web UI（Agent 选择器）
- 记忆系统（智能摘要、长期记忆）
- 技能系统（Office/Outlook）
- Playwright Web 自动化

### 重要命令
```bash
h-agent init                    # 初始化
h-agent start/stop/status       # 后台服务
h-agent team init               # 初始化团队
h-agent team talk <agent> "msg" # 与特定 agent 对话
h-agent web                     # 启动 Web UI
```
