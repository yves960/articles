# Raw Sources

这里存放知识库的原始材料。既有文章已经迁入这里，后续新增原始资料也放在这里。

当前主要内容：

- `*.md`：既有独立文章源材料。
- `ai-replacement-series/`：AI 替代/基础设施系列源材料。
- `ai-assisted-dev-specs/`：AI 辅助研发规范体系源材料。

也适合继续添加：

- 访谈转录
- 网页摘录
- 论文笔记原文
- 会议记录
- 未整理的长文本

规则：

- 默认不要改写 raw 文件。
- 如果需要修正 OCR、转录错误或格式问题，保留原始含义。
- 每个 raw 文件整理进 wiki 后，在 `../wiki/source-map.md` 增加映射。
